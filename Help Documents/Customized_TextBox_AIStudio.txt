You're absolutely right, CornerRadius is not a direct property 
of a TextBox. You've also hit on the common problem when trying 
to wrap a TextBox in a Border with CornerRadius: the TextBox's 
default template usually has its own square Border and Background 
that simply covers your rounded Border.

To get rounded corners for a TextBox, you have to tackle its 
ControlTemplate, similar to what we did with the ComboBox. You 
need to find the Border element within the TextBox's default 
template and apply CornerRadius there.

Here's how to do it using a Style and ControlTemplate in a 
ResourceDictionary, which is the most robust and reusable method.

Steps to Create a Rounded Corner TextBox Style

1. Create Styles/RoundedTextBox.xaml

Create a new XAML file in your Styles folder (e.g., Styles/RoundedTextBox.xaml).


<!-- Styles/RoundedTextBox.xaml -->
<ResourceDictionary xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">

    <!-- Define your common dark theme colors (if not already defined in App.xaml merged dictionaries) -->
    <SolidColorBrush x:Key="DarkBackgroundBrush" Color="#3F4347"/>
    <SolidColorBrush x:Key="LightForegroundBrush" Color="#DBDBDB"/>
    <SolidColorBrush x:Key="MediumBorderBrush" Color="#555"/>
    <SolidColorBrush x:Key="TextBoxMouseOverBorderBrush" Color="#7A7A7A"/>
    <SolidColorBrush x:Key="TextBoxFocusedBorderBrush" Color="#007ACC"/> <!-- Example accent color for focus -->
    
    <!-- ============================================================================ -->
    <!-- Style for TextBox with Rounded Corners                                       -->
    <!-- ============================================================================ -->
    <Style x:Key="RoundedTextBoxStyle" TargetType="{x:Type TextBox}">
        <!-- Default properties for your dark theme TextBox -->
        <Setter Property="Background" Value="{StaticResource DarkBackgroundBrush}"/>
        <Setter Property="Foreground" Value="{StaticResource LightForegroundBrush}"/>
        <Setter Property="BorderBrush" Value="{StaticResource MediumBorderBrush}"/>
        <Setter Property="BorderThickness" Value="1"/>
        <Setter Property="Padding" Value="5"/>
        <Setter Property="CaretBrush" Value="{StaticResource LightForegroundBrush}"/>
        <Setter Property="SnapsToDevicePixels" Value="True"/>
        <Setter Property="VerticalContentAlignment" Value="Center"/>

        <!-- The ControlTemplate where we apply the CornerRadius -->
        <Setter Property="Template">
            <Setter.Value>
                <ControlTemplate TargetType="{x:Type TextBox}">
                    <Border x:Name="Bd" 
                            Background="{TemplateBinding Background}"
                            BorderBrush="{TemplateBinding BorderBrush}"
                            BorderThickness="{TemplateBinding BorderThickness}"
                            CornerRadius="5" > <!-- KEY FIX: Apply CornerRadius here -->
                        <ScrollViewer x:Name="PART_ContentHost" 
                                      SnapsToDevicePixels="{TemplateBinding SnapsToDevicePixels}"
                                      Focusable="False" 
                                      HorizontalScrollBarVisibility="Hidden" 
                                      VerticalScrollBarVisibility="Hidden"/>
                    </Border>
                    <ControlTemplate.Triggers>
                        <!-- Change border color on mouse over -->
                        <Trigger Property="IsMouseOver" Value="True">
                            <Setter TargetName="Bd" Property="BorderBrush" Value="{StaticResource TextBoxMouseOverBorderBrush}"/>
                        </Trigger>
                        <!-- Change border color on focus -->
                        <Trigger Property="IsKeyboardFocused" Value="True">
                            <Setter TargetName="Bd" Property="BorderBrush" Value="{StaticResource TextBoxFocusedBorderBrush}"/>
                        </Trigger>
                        <!-- Placeholder text visibility (if you use it) -->
                        <Trigger Property="Text" Value="">
                            <Setter Property="Foreground" Value="#888" TargetName="PART_ContentHost"/>
                        </Trigger>
                        <Trigger Property="IsEnabled" Value="False">
                            <Setter Property="Opacity" Value="0.5"/>
                        </Trigger>
                    </ControlTemplate.Triggers>
                </ControlTemplate>
            </Setter.Value>
        </Setter>
    </Style>

</ResourceDictionary>

Explanation of Changes:

Border x:Name="Bd": This is the key element within the default 
TextBox template that forms its visual border and background.

CornerRadius="5": This line is added to the Border named Bd. 
This applies the rounding directly to the element that forms 
the visible box of the TextBox. You can adjust 5 to any value 
you like for more or less rounding.

ScrollViewer x:Name="PART_ContentHost": This is the actual 
area where the text content of the TextBox is displayed. We 
don't want to apply CornerRadius directly here, as it's the 
internal content host.

Triggers: I've included common triggers to change the BorderBrush on IsMouseOver and IsKeyboardFocused to provide visual feedback to the user, which is good UX, especially for text input fields.

2. Update App.xaml to Merge the New Resource Dictionary

Add this new RoundedTextBox.xaml to your App.xaml's MergedDictionaries:

<!-- App.xaml -->
<Application x:Class="YourAppName.App"
             xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
             xmlns:local="clr-namespace:YourAppName"
             StartupUri="MainWindow.xaml">
    <Application.Resources>
        <ResourceDictionary>
            <ResourceDictionary.MergedDictionaries>
                <ResourceDictionary Source="Styles/DarkThemeComboBox.xaml"/>
                <ResourceDictionary Source="Styles/DarkThemeScrollBar.xaml"/>
                
                <!-- NEW: Merge your custom TextBox style dictionary -->
                <ResourceDictionary Source="Styles/RoundedTextBox.xaml"/>
                
            </ResourceDictionary.MergedDictionaries>
        </ResourceDictionary>
    </Application.Resources>
</Application>

3. Apply the Style in Your View (MainWindow.xaml or similar)

Now, in any XAML file where you use a TextBox, you can apply this style:


<!-- YourView.xaml -->
<Window x:Class="YourAppName.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Your Application" Height="450" Width="800">
    <Grid Background="#2A2A2A">
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="Auto"/>
        </Grid.RowDefinitions>
        <Grid.ColumnDefinitions>
            <ColumnDefinition Width="Auto"/>
            <ColumnDefinition Width="10"/>
            <ColumnDefinition Width="*"/>
        </Grid.ColumnDefinitions>

        <TextBlock Grid.Row="0" Grid.Column="0" Text="Client Name:" VerticalAlignment="Center" Foreground="#DBDBDB" Margin="0,20,0,10"/>
        <TextBox Grid.Row="0" Grid.Column="2" 
                 Text="Example Client"
                 Style="{StaticResource RoundedTextBoxStyle}" 
                 Margin="0,20,0,10"/>

        <TextBlock Grid.Row="1" Grid.Column="0" Text="Address:" VerticalAlignment="Center" Foreground="#DBDBDB" Margin="0,10"/>
        <TextBox Grid.Row="1" Grid.Column="2" 
                 Text="123 Wellness Rd"
                 Style="{StaticResource RoundedTextBoxStyle}" 
                 Margin="0,10"/>
        
        <!-- Your ComboBox example -->
        <TextBlock Grid.Row="2" Grid.Column="0" Text="Marital Status:" VerticalAlignment="Center" Foreground="#DBDBDB" Margin="0,20,0,10"/>
        <ComboBox Grid.Row="2" Grid.Column="2" 
                  Text="Married"
                  Margin="0,20,0,10" 
                  Style="{StaticResource MyCustomComboBoxStyle}" 
                  >
            <ComboBoxItem Content="Single"/>
            <ComboBoxItem Content="Married"/>
            <ComboBoxItem Content="In Relationship"/>
            <ComboBoxItem Content="Divorced"/>
            <ComboBoxItem Content="Widowed"/>
        </ComboBox>
    </Grid>
</Window>

Now, your TextBox controls that use RoundedTextBoxStyle will display 
with beautifully rounded corners, consistent with your 
dark theme.

To achieve this, the best approach in WPF is to create a Style 
with a custom ControlTemplate. This avoids creating a whole new 
UserControl and allows you to keep using the standard TextBox 
functionality (like Bindings) while completely changing how it looks.

Here is the solution broken down into two parts:

The Style (Resources): You can put this in App.xaml (to use globally) or 
UserControl.Resources.

The Implementation: How to update your Treatment.xaml to use it.


Part 1: Define the Style

Add this code to your <UserControl.Resources> section (or App.xaml).

Key Implementation Details:

Rounded Corners: We wrap the content in a Border with CornerRadius="15".

Icon: We use a Path geometry inside the template grid.

Placeholder: We place a TextBlock over the input area. A Trigger checks if the Text property is empty; if it is, the placeholder is shown.

Flexible Text: We bind the Placeholder text to the Tag property of the TextBox, allowing you to change the placeholder per view.


<UserControl.Resources>
